CJLDialogBox

Light-weight,customizable and easy to use Modal/Dialog Box
[Download (4kb)]

Installation

Requires jQuery 1.6+

<link rel="stylesheet" href="cjlDialog.css">
<script src="jquery-1.10.2.min.js"> </script>
<script src="cjlDialog.js"> </script>

Basic Usage

Simple/Plain DialogBox

DEMO
<script> $.cjlDialog({ title:'myTitle', content:'Lorem ipsum' }); </script>

DialogBox with Type

DEMO
<script> $.cjlDialog({ title:'myTitle', content:'Lorem ipsum', overlay:true, type:' ' }); </script>

With imaga & embed Video

Image Video
<script> $.cjlDialog({ title:'myTitle', content:'<img src="sample.jpg" width=600 height=450>', contentCss:{ width:600, height:450 }, overlay:true }); </script>

API

Options & Callbacks
  • title [String] required
    Sets the title of the dialogbox
  • content [String] required
    Content of dialogbox
  • type [String]
    Alert type: info, success, error, warning, ask
  • contentCss [ Object {width:350,height:100,padding:20} ]
    Sets the content style
  • overlay [boolean:false]
    If true, a backdrop will be added
  • position [Object:{top:n,left:n}] or [function($container)]
    Sets the container's Location (by default, the position uses a function type)
    if Object, property top and left will sets the container's location
    if function, the main container's jquery object will be pass, for you to customize
  • buttons [ Array:[] ]
    buttons is an array of Object,commonly with property of text,attr and on (the on events..like onclick,onmouseenter,etc)
    See the advance section for an example..
  • transitionIn [ Object: {style:'fadeIn'} ] or [function($container,afterShow)]
    In customizing the transition, the style property is a requierd one.
    Style property should be a valid animation style, ex. (fadeIn,fadeOut,show,slideDown,etc),you can also set any animation properties like duration,effect,easing,etc..
    example (transitionIn:{style:'slideDown',duration:1000,easing:'easeOutBounce'} )

    if function, it will pass the container jquery object and afterShow method
    Note: Don't forget to call the afterShow() argument.
  • transitionOut [ Object: {style:'fadeOut'} ] or [ function($container,afterClose) ]
    See transitionIn..
    See Advance Usage..
  • onShow [ function($container) ]
    Triggers before show... You can just ignore the $container if you want to.
  • afterShow [ function($container) ]
    Triggers after the container is showed
  • onClose [ function($container) ]
    Triggers before the container is being close
  • afterClose [ function($container) ]
    Triggers after the container is close

Advance Usage

Position

DEMO
<script> $.cjlDialog({ title:'myTitle', content:'Lorem ipsum', position:{top:0,left:500} }); </script>

Buttons

DEMO
<script> $.cjlDialog({ title:'myTitle', content:'Lorem ipsum', buttons:[{ text:'Click Me', attr: { id:'btn1', class:'btn' }, on:{click: function(e){ alert('You Clicked Me :)') } } }, //end button1 text:'Hover Me', attr: { id:'btn2', class:'btn' }, on:{mouseenter: function(e){ alert('Mouse enter') } } }, //end button2 ] }); </script>

Transitions

DEMO
<script> $.cjlDialog({ title:'myTitle', content:'Lorem ipsum', transitionIn: {style:' ', duration: } transitionOut: {style:' ', duration: } }); </script>

Transitions with jQueryUI Effects

DEMO
<script> $.cjlDialog({ title:'myTitle', content:'Lorem ipsum', transitionIn: {style:' ', effect:'', direction:'', easing:'', duration: } transitionOut: {style:' ', effect:'', direction:'', easing:'', duration: } }); //direction,easing,duration are optional //you can use any valid animation properties </script>

Modal

It's not just a dialogbox, but can also be a modal. $.cjlDialog and $(el).cjlModal uses the same functionality (except for title & content property)

Public Methods

  • open()
  • close()

HTML

<div id="myModal"> <h1>myModal Title</h1> <div>Lorem ipsum....</div> </div>

Javascript

DEMO
<script> var myModal; // for global use $(document).ready(function(e){ myModal = $('#myModal').cjlModal({ transitionIn:{style:'show', effect:'explode'} }); // with transitionIn explode and fadeOut transitionOut(default) //you can now use the myModal.open() and myModal.close() }); </script>
